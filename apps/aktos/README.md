# Aktos App

# Development

### Linux

To start WebUI development:

```bash
./dev-ui
```

Then visit [http://localhost:4002](http://localhost:4002)

To start embedded development:

```bash
./aktos/dev-nfc
```

### Windows

* Open `git BASH`
* `gulp --project=aktos`
* `cd apps/aktos/webserver; lsc server.ls`
