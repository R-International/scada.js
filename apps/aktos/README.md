# Aktos App

# Development

### Linux

To start WebUI development:

```bash
{{ scada }}/apps/aktos/dev-ui
```

Then visit [http://localhost:4002](http://localhost:4002)

To start Embedded system development:

```bash
{{ scada }}/apps/aktos/dev-nfc
```

### Windows

* Open `git BASH`
* `gulp --project=aktos`
* `cd apps/aktos/webserver; lsc server.ls`
