# Contributing

## Reporting/Fixing Bugs 

1. Reproduce a bug in the [`template project`](https://github.com/aktos-io/scadajs-template)
2. Create a PR with a name like "Reproduction of bug #XXXX". 
3. File the issue 
4. Fix the issue 
5. Open a PR.
